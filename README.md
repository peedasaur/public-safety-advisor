Public Safety Advisor is a simple web application that provides basic safety guidance during emergency situations. The main purpose of this project is to offer quick and clear advice when someone is unsure about what steps to take in a risky or urgent situation. The application focuses on keeping the interface minimal, easy to understand, and accessible on both desktop and mobile devices.

About the Project

This project is designed to give users situation-based safety suggestions along with basic emergency response guidance. The interface is clean and straightforward so that users can quickly find the information they need without confusion. The overall structure of the project is kept simple to make it easy to understand and maintain.

Technologies Used

The project is built using HTML, CSS, and JavaScript for the frontend. If a backend is included, it uses Node.js and Express. MongoDB is used if database functionality is implemented.

How to Run the Project

First, clone the repository using the following command:

git clone https://github.com/peedasaur/public-safety-advisor.git

Then move into the project folder:

cd public-safety-advisor

If the project includes a backend, install the required dependencies:

npm install

After that, start the server using:

npm start

If the project is frontend-only, you can simply open the index.html file in your browser.

Future Improvements

In the future, the project can be improved by adding real-time emergency alerts, enhancing the user interface, implementing user authentication, and deploying the application to a cloud platform.
